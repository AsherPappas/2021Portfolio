Project 1: Best Snow

GitHub Repository

This app will use the Powderlines API to retrieve snow data for requested SNOTEL stations. The Bing Maps API then shows the SNOTEL stations on a map with snowflake icons representing each station. Once you select an icon, a graph will appear showing the current snow depth plus the snow depth data over the last 5 years.
